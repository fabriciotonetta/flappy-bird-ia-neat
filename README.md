# 🐦 Flappy Bird com Inteligência Artificial (NEAT)

> IA que aprende a jogar Flappy Bird sozinha, evoluindo através de algoritmos genéticos (NEAT) e redes neurais artificiais.

## 🎯 Sobre o projeto

Este projeto recria o clássico jogo Flappy Bird e treina uma população
de redes neurais para jogar de forma autônoma, sem nenhuma programação
manual de "regras de como jogar". A IA aprende por tentativa, erro e
seleção natural simulada — exatamente como na evolução biológica.

## 🛠️ Tecnologias utilizadas

- Python
- Pygame (motor do jogo)
- NEAT-Python (algoritmo genético / redes neurais evolutivas)
- Google Colab (ambiente de desenvolvimento)
